> This is a small landing page for some fictional company.

## Install

```
$ git clone ...
$ npm install
```

## Usage

```
$ npm run dev
```

and open http://localhost:1234 in your browser, or

```
$ npm run build
```

and deploy somewhere your new **dist** folder.
